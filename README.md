# Kanban Board

A lightweight Kanban Board application built with HTML, CSS, and vanilla JavaScript for organizing and managing tasks. The project features drag-and-drop functionality, task editing, local storage persistence, and a clean modular architecture.

---

## Overview

Kanban Board is a task management application that helps users organize their workflow using the Kanban methodology. Tasks can be created, updated, deleted, reordered, and moved between different workflow stages through an intuitive drag-and-drop interface.

---

## Features

* Create New Tasks
* Edit Existing Tasks
* Delete Tasks
* Drag & Drop Task Management
* Move Tasks Between Columns
* Reorder Tasks Within Columns
* Automatic Data Persistence using Local Storage
* Clean and Responsive User Interface
* Modular JavaScript Architecture

---

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6 Modules)
* Local Storage

---

## Workflow

The board is divided into three workflow stages:

* Not Started
* In Progress
* Completed

Users can easily move tasks between stages as their work progresses.

---

## Key Highlights

* Built entirely with vanilla JavaScript
* Object-oriented and modular code structure
* Interactive drag-and-drop functionality
* Persistent task storage with Local Storage
* Lightweight and dependency-free implementation
* Simple and intuitive user experience
